Doorstate
=========

Complete rewrite of doorstate in [Rust](https://www.rust-lang.org/).

Currently WIP.

Requirements
------------

### Client

- [Rust](https://www.rust-lang.org/)
- [Cargo](https://crates.io/)

#### Mac OS X (Homebrew)

`brew install rust`

#### Gentoo

`emerge dev-lang/rust dev-util/cargo`

### Server

- [PHP](https://secure.php.net/)
